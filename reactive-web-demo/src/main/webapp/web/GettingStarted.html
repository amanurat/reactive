Getting Started with reactive-web

  1. Choose an existing Lift project or create a new one

  2. Make sure you have the Sonatype maven repository configured

    For instance, if you're using sbt, that would be:

      resolvers += "Sonatype snapshots" at "http://oss.sonatype.org/content/repositories/snapshots"

  3. Add reactive-web as a dependency

    For instance, if you're using sbt, that would be:

      libraryDependencies += "cc.co.scala-reactive" %% "reactive-web" % "0.3.2.1"

  4. Enable it

    1. In Boot.scala:

        Reactions.init()

      To enable comet support, use

        Reactions.init(true)

    2. On each page (you can put it in your surround template, e.g., templates-hidden/default.html), add:

        <div class="lift:reactive"></div>

  5. Start coding!

    • You will probably want to import reactive._ and reactive.web._
    • Snippets may need to extend Observing
